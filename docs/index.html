workerinstall - a package manager

workerinstall is a simple, but efficient, package manager.
Installing is as simple as 

            worker
Worker's usage is: 
Usage: worker [-pqsuUv?V] [-P DIR] [--purge] [--prefix=DIR] [--quiet]
            [--silent] [--upgrade] [--update] [--verbose] [--help] [--usage]
            [--version] package(s)

workerinstall comes preinstalled in the doomlinux linux distribution, but can be used in other distributions,
if you do it is recommended you use this in a distribution without a package manager.

Installing workerinstall can be done with: 

git clone https://github.com/DOOMLinux/workerinstall.git
make
make install
